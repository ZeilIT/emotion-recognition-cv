# Распознавание эмоций по лицу (Computer Vision)

Проект для определения эмоции человека на фотографии с помощью глубокого обучения.  
Поддерживаемые эмоции: **гнев, отвращение, страх, радость, нейтрально, грусть, удивление**.

---

## 🔗 Репозиторий с кодом
[https://github.com/ZeillT/emotion-recognition-cv](https://github.com/ZeillT/emotion-recognition-cv)

---

## 📦 Файл весов модели

В проекте используется библиотека **DeepFace**, которая **автоматически загружает предобученные веса** при первом запуске (модели VGG, FaceNet и др.).  
- Веса сохраняются локально в папку `~/.deepface` (объём ~500 МБ).  
- Пользователю **не нужно вручную скачивать** веса – они загрузятся сами при выполнении кода (см. инструкцию ниже).  
- Скрипт загрузки весов встроен в библиотеку DeepFace и не требует отдельного файла.

## 🧪 Инструкция по воспроизведению результата (Inference)

### ✅ Рекомендуемый способ – Google Colab (всё работает в браузере)

1. **Откройте блокнот в Colab** по ссылке:  
   [https://colab.research.google.com/github/ZeillT/emotion-recognition-cv/blob/main/emotion_recognition.ipynb](https://colab.research.google.com/github/ZeillT/emotion-recognition-cv/blob/main/emotion_recognition.ipynb)

2. **Включите GPU** (ускорит работу):  
   Меню → **Среда выполнения** → **Сменить среду выполнения** → выберите `T4 GPU` → **Сохранить**.

3. **Запустите ячейки по порядку** клавишами `Shift+Enter`.

4. **Результат**: текст с эмоцией и картинка с рамкой.

### 📌 Примечания
- Лицо должно быть анфас, хорошее освещение.
